# Dinaya Competitive Positioning (Sri Lanka)

## Local booking competitors

| Player | Their angle | Dinaya wedge |
|--------|-------------|--------------|
| OrderNow Bookings | Broad verticals, 500+ businesses | Sharper positioning + concierge onboarding |
| Servio.lk | Consumer marketplace, no-account booking | Link-first public pages + Deals discovery |
| Linking.lk | App + directory | Simple shareable link everywhere |
| Glamr / BeauTech / Bookr | Beauty ops, POS, payroll | SL-local, web-link-first, PayHere, growth workflows |

## Global tools (not primary competitors)

| Player | Gap vs Dinaya |
|--------|---------------|
| Calendly / Fresha / Booksy | Not built for SL payments, WhatsApp habits, local setup |

## Ticketing (events expansion)

| Player | Segment | Dinaya angle |
|--------|---------|--------------|
| Spotseeker | EDM/nightlife discovery | Defer head-on; host-branded SME events first |
| PickMe Events | Stadium mega-shows | Don't compete early |
| Ticket Vault, Eventhub | Thin inventory | Win on product + CRM + transparent fees |
| EventsHere | 7% flat, workshops | Same PayHere stack + vertical bundle |

## How to compete (approved framing)

- Avoid feature-for-feature with global tools
- Win on: **local setup, PayHere, WhatsApp/SMS, growth workflows, founder proximity**
- Generic appointment booking is **crowded** — need sharper positioning
- Pitch: **faster, clearer, more trustworthy for Sri Lankan businesses**

## Market context

- SL event industry ~Rs 60B (2025); mobile-first; social = demand channel
- SMEs = 75%+ of companies; salons market visually on IG/FB/WhatsApp
- One dense geographic cluster before multi-city expansion
